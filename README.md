# RAG-бот для вопросов по документации

Бот позволяет загружать PDF/TXT файлы и задавать вопросы по их содержимому с автоматическим цитированием источников.

## Технологии
- LangChain 0.1.0
- FAISS (векторная БД)
- Ollama + Llama 3.2
- Streamlit

## Возможности
- Загрузка PDF и TXT файлов
- Разбивка на чанки с перекрытием
- Поиск по смыслу (векторный поиск)
- Генерация ответов с цитированием
- Локальная работа (без API-ключей)
